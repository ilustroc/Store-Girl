# Frontend Bootstrap - Tecno Store

Interfaz del ecommerce con Bootstrap 5, catalogo conectado al backend Java y panel administrativo.

## Ejecutar

Desde `frontend`:

```powershell
node dev-server.js
```

Abrir:

```text
http://localhost:5500
```

El backend debe estar activo en `http://localhost:8080/api`.
