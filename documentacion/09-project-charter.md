# 09. Project Charter

| Campo | Detalle |
| --- | --- |
| Proyecto | TecnoStore |
| Objetivo | Crear un ecommerce tecnologico funcional para exposicion universitaria. |
| Alcance | Frontend con rutas, backend Spring Boot, base normalizada, CRUD de productos, carrito, checkout y pedidos. |
| Fuera de alcance | Modulo de reportes, pasarela de pagos real, seguridad productiva. |
| Stakeholders | Equipo de proyecto, docente, usuarios de prueba. |
| Tecnologias | HTML, CSS, JavaScript, Bootstrap 5, Spring Boot, JPA, H2. |
| Riesgos | Falta de configuracion Java/Maven, errores de CORS, datos inconsistentes. |
| Criterio de exito | Demo local funcional con usuario y administrador. |
