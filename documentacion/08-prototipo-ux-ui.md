# 08. Prototipo UX/UI

## Vistas implementadas

- Home.
- Catalogo.
- Detalle de producto.
- Carrito.
- Checkout.
- Login.
- Registro.
- Mis pedidos.
- Admin.
- Admin productos.

## Criterios visuales

El diseno toma como referencia patrones comunes de ecommerce tecnologico:

- Barra superior con buscador.
- Hero comercial.
- Cards de producto con imagen, categoria, stock, precio y accion.
- Detalle de producto con imagen grande.
- Carrito con cantidades y resumen.
- Checkout simple.
- Panel administrador con tabla.

No se copia ningun diseno exacto de tiendas reales.
