# 01. Contexto del problema

TecnoStore es un prototipo de tienda online para productos tecnologicos. El proyecto busca resolver la desorganizacion inicial del frontend y convertirlo en una solucion web clara, con backend en Java, base de datos y documentacion tecnica.

## Problema

La version inicial concentraba varias responsabilidades en una sola vista y dependia de datos locales. Esto dificultaba el mantenimiento, la escalabilidad y la exposicion academica.

## Solucion propuesta

Construir una arquitectura separada:

- Frontend con Bootstrap 5 y rutas dinamicas.
- Backend Spring Boot con API REST.
- Base de datos normalizada.
- Roles de administrador y usuario.
- Carrito, checkout, pedidos y CRUD de productos.

No se incluye modulo de reportes en esta etapa.
